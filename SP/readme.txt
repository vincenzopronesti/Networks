La directory data/ contiene i file relativi ai peer associati. 
Il nome dei file e' l'indirizzo IP del peer, il contenuto e' la lista 
dei file condivisi. 

La directory files/ contiene l'insieme dei file condivisi. Si tratta dei file che, 
o sono gia' presenti all'avvio oppure vengono aggiunti alla directory dopo essere 
stati scaricati.

Il file availableSP.txt contiene gli indirizzi IP degli altri superpeer, 
forniti dal nodo di bootstrap.
